Repositorio personal de estudio y referencia para entrevistas técnicas relacionadas con Inteligencia Artificial, APIs, Python, SQL Server, automatización y desarrollo de soluciones empresariales.



\## Objetivo



Este repositorio reúne ejemplos prácticos, plantillas de código y apuntes técnicos utilizados para reforzar conocimientos en:



\- Inteligencia Artificial aplicada a negocio

\- Consumo de APIs REST

\- Python

\- SQL Server

\- Automatización de procesos

\- Integración Python + SQL

\- Docker básico

\- MongoDB básico

\- ISO 27001

\- Casos prácticos de entrevistas técnicas
